# Collection HAL CAA-FR 

Suite à ACQuA 2026, une [collection](https://hal.science/CAA-FR) a été créée sur HAL afin de regrouper les posters des communicant·e·s qui voudraient les déposer. Les admin sont : Sébastien Plutniak, Salomé Mega, Julie Gravier. 
 
Cette collection pourrait être organisée pour faciliter tout futur dépôt en lien avec les activités du chapitre, en vérifiant par exemple la possibilité de créer des sous-catégories (tags, sous-collections... ) [Seb. P./Salomé]. 
 
Se pose notamment la question de revenir vers les communicant·e·s des posters de l'année précédente pour leur proposer de déposer leur poster dans la collection [Séb].

## Débat : la publication d'actes dans *PCI Archaeology* 
 
Question déjà soulevée à la dernière réunion du CAA-FR (18/02) ; à rediscuter avec plus de monde. 
 
Discussion sur les difficultés rencontrées pour collecter les actes d'ACQuA 2026 :  
 
* Actes des communications : un point pourrait être fait avec les communiquant·e·s au moment d'ACQuA pour se renseigner sur leur volonté de déposer (ou non) des actes de leur communication. 
* Table ronde 2026 : un premier texte commun pourrait être rédigé comme récapitulatif de la table ronde. Il faut relancer les auteur.trice.s pour pouvoir ensuite envisager une publication [mettre Romane dans la boucle mail : Séb P.]
